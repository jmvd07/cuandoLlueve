Antes de ejecutar el código "Cuando Llueve.inpy" es preciso tener instalados los
Drivers de Python necesarios para conectarse a la base de datos de MySQL. Para esto,
se recomienda aplicar el siguiente comando en el Terminal de Linux o la CMD de Windows:

pip install mysql-connector-python

PRECAUCIÓN: Algunos Antivirus o administradores de internet pueden tener Corta-Fuegos
(o FireWall) que impidan la conexión a la base de datos remota de MySQL. Si habiendo
instalado el complemento de mysql no puede conectarse a la base de datos, revise su
configuración de FireWall.
